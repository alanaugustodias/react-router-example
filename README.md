# Getting started

## Installing

### Global dependencies

- Node:

[Documentation](https://nodejs.org/en)

```bash
brew install node
```

- Pnpm:

[Documentation](https://pnpm.io/motivation)

```bash
corepack enable pnpm

corepack use pnpm@latest
```

### Project dependencies

```bash
pnpm install
```

## Running

### Build

```bash
pnpm build
```

### Lint

```bash
pnpm lint
```

### Dev

```bash
pnpm dev
```
